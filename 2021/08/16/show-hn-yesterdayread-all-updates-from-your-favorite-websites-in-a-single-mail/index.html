Show HN: Yesterday-Read all updates from your favorite websites in a single mail

Nokuma 2021-08-16 14:15:45 +0000 UTC [ - ]

Hi, Yesterday is a newsletter service where you can get updates from all your favorite websites/ blogs and read them in one single e-mail. I've been using RSS for a long time but I wanted some kind of tool to get updates from sites that doesn't support RSS as well (plus I'm not a fan of newsletter as it clutters my inbox). I created Yesterday so I'll know what's been updated on each site with the simplicity of reading the contents in one simple email format. Yesterday will send daily newsletter at 8am. It has freed me from checking websites and the worry of missing updates. I'm pretty happy with what I've built and thought of sharing it here. Leave me comments or feedbacks! thanks
