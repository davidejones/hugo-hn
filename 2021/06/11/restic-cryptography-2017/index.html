justin_oaks 2021-06-13 04:57:30 +0000 UTC [ - ]

Restic is pretty great. I use it for backups at work and my personal backups too. The fact that it's encrypted makes it so I can use cloud storage without worrying about what happens if (when?) the cloud provider gets hacked. The reduplication allows me to have both incremental and full backups in one application. If you're comfortable on the command line then I recommend Restic without reservation.

If you're not comfortable on the command line then you may have to reach for another tool.

I've used SyncThing. It's good for keeping folders on multiple devices in sync, but they are NOT backups. Backups allow you to recover from accidental or malicious deletions and modifications. SyncThing will happily move those deletions and modifications to your other systems, so both are in-sync and both are wrong.
