rav 2021-06-13 08:31:01 +0000 UTC [ - ]

Simple scheme for deduplicating strings in Python:

    dedup = {}
    for s in strings:
        s = dedup.setdefault(s, s)
        # Do something with s
It hashes each string "s" and looks in the dictionary "dedup" to see if there's an existing string object previously in the list "strings", in which case it uses it, and otherwise it stores the current string object in "dedup".

masklinn 2021-06-13 09:55:16 +0000 UTC [ - ]

Simpler scheme for deduplicating strings in Python: use `sys.intern`. It also specifically increases hash performances because interned strings can be compared by address after a hash collision (and furthermore hash caching means interned strings almost certainly already have their hash computed and stored)

    for s in strings:
        s = sys.intern(s)
        # Do something with s
Depending on your application lifecycle you may want to shove the dedup'd strings in a set (or in a list, after checking if the identity of what you got from `sys.intern` is different than what you put in) in order to keep them alive: `sys.intern` does not do that.
